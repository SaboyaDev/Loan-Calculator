# Loan Calculator - JS

A JavaScript loan calculator built with Bootstrap

### Enter: 
- Amount to borrow
- Interest Rate
- Length ofloan

### Expected Results:
- Monthly payments
- Total interest payed
- Grand Total (Interest Payed + Money Borrowed)
